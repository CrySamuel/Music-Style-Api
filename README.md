# Music-Style-Api

Hello, this API was created to talk about musical styles and also to develop my skills with Spring Boot.

It is separated by 4 folders, controller where the methods will work via Http using postmain, model folder where the main definition of the class of its variables is, etc. Repository this folder contains an Interface type file where I create a method to search for items with the same name or another type and finally resources where the application.yml file is which is the connection to the database in H2 storage and the SQL code of the database.

Currently the only tools and libraries I currently use are postmain, Spring Boot DevTools, Spring Web, Spring Data JPA and H2 Database.
